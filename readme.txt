== Description ==

Show your posts activity using a little progress image. Activity is based on comments/views ratio. Easy setup through admin panel.

Features:

* Easy install and setup
* Use `wp_post_activity()` function in your theme to output the activity image.
* You can customize the look of the activity image (look thru the images folder)
* Still in development, accepting feature requests!

== Installation ==

1. Upload archive contents to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Check the settings for WP Post Activity and set your highest/lowest barriers (usually 0.5 - 0.01 respectively)
1. Show the image in your theme via the wp_post_activity() function. Just call it.
